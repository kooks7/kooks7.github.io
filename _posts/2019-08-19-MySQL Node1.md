### 생활코딩 DATABASE2 MySQL 수업 듣기





### 1. 스키마의 사용

* 데이터베이스 생성   
  `mysql> CREATE DATABASE test;`  
* 데이터베이스 삭제  
  `mysql> DROP DATABASE test;`  
* 데이터베이스 보기   
  `mysql> SHOW DATABASES;`  
* 데이터베이스 사용  
  `mysql> USE test;`

### 3.테이블의 생성

* 실습

  ```mysql
  mysql> USE test;

  	-> id INT(11) NOT NULL AUTO_INCREMENT, //필수이고 자동으로 생성
  ```
